# Market Vocabulary Agent

## Overview
Market Vocabulary Agent is an educational AI application that helps users learn financial and market terminology from daily Bloomberg news.

## Features

- Daily vocabulary extraction
- AI-generated explanations
- Quiz generation
- Markdown lesson export
- JSON lesson export
- Gemini AI integration
- Dry-run mode
- Structured Pydantic schema

## Project Structure

app/
data/
requirements.txt

## Installation

pip install -r requirements.txt

## Usage

python -m app.main --dry-run
